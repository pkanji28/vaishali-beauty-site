
Vaishali's Beauty - Full Production ZIP
=======================================

What is included
----------------
- Premium homepage
- Booking page with:
  - Categorised services
  - Quantity + / - controls
  - Live total
  - Notes field
  - Date-first booking flow
  - Collision checking against saved bookings
  - WhatsApp confirmation message
- Feedback page
- Admin page with:
  - PIN login
  - Booking deletion
  - Feedback approval / deletion
  - Gallery image upload / deletion

Important
---------
This build works fully in the browser using localStorage.
That means:
- it works immediately after upload
- data is stored in the same browser/device
- if you later want shared online admin data, that is the next backend build

Default admin PIN
-----------------
1234

You can change it in:
js/config.js

Files to upload
---------------
Upload ALL files and folders from this ZIP to your repository.

Main files:
- index.html
- book.html
- feedback.html
- admin.html
- css/styles.css
- js/config.js
- js/data.js
- js/storage.js
- js/book.js
- js/feedback.js
- js/admin.js

Recommended upload method
-------------------------
1. Open your GitHub repository
2. Delete old duplicate files if needed
3. Upload everything from this ZIP
4. Commit changes
5. Wait for Vercel to deploy
6. Hard refresh your browser

Locking baseline
----------------
Once you confirm this build is working, use this as the locked baseline.
Future changes should be incremental improvements only.
